# :raised_hands: Mybest
Mybestは**モノに対する判断軸**を見つけるための、所有物の管理、共有するサービスです。  
衣類、本、雑貨の3種類を登録できますが、登録数はそれぞれ制限があります。  
登録する過程で自分と対話し、**モノに対する判断軸**を見つける旅に出ましょう！

# :file_folder:制作の背景

私は断捨離が趣味です。

そんな私ですが、1年前はいわゆる汚部屋の住人でした。  
しかし、今は生活にミニマリズムを取り入れ、自分の好きなモノに囲まれて過ごしています。  

【現在の自分の部屋】
<img width="800" src="https://user-images.githubusercontent.com/67394974/99733961-55d84600-2b05-11eb-99a7-45dc39efd0c3.jpeg">

断捨離ができなかった頃の自分の課題は、以下の通りでした。  

* __「いつか使うだろう」と捨てることを後回しにしてしまう__
* __モノを捨てるが続かない、すぐに元通りになってしまう__
* __「セールだ！」とモノを余分に買い込んでしまう__　

なぜ断捨離ができなかったのでしょうか？  

**それは、「何を残すか」ということを決めていなかったからです。**  
つまり、自分の最適なモノの**判断軸**がフワフワしたままになっていたのです。  
また、**判断軸**が明確になった時、「こだわって選んだモノを皆に見せたい！」と感じました。  

以上の経験を踏まえ、**モノに対する判断軸を見つけ、残したお気に入りのモノを共有するサービス**があれば上記の課題を解決できるのではないかと思い、本アプリを制作しました。
  
# :relaxed:アプリを利用するメリット
このアプリを使用することで以下のメリットが得られます。

### モノが減る
* モノを探す時間、管理する時間が減り、**本当にやりたいことに時間を費やせる**
* 意思決定の機会が減り、**集中力が上がる**
* 維持コストや無駄な出費が減り、**節約になる**

### モノの減らし方がわかる
* 自分のモノに対する**判断軸が見つかる**
* **お気に入りのモノ**に囲まれて生活できる

# :dizzy:工夫した点
* モノに対する判断軸を見つけるために、あえて**投稿数の制限**を設けました。
* 「モノをなぜ残すのか」を考えるために、**質問形式**の投稿フォームにしました。
* 人によって登録数がオーバーしてしまう可能性があるため、制限付きで**増枠**できるようにしました。（max10枠まで）
* 「**直感的なわかりやすさ・使いやすさ**」 を重視し、一つ一つの機能やデザインに細部までこだわりました。
* **ミニマリズム感**を意識するために、色は3色以内で余白を取ったデザインにしました。

# :green_book:機能一覧
### ユーザー管理機能
* ユーザー登録・編集・削除（devise）
* ゲストログイン
* プロフィール画像の登録・編集（Active Strage)
* マイページに以下の投投稿の一覧表示
  * すべての自分の投稿
  * 自分がピックアップした投稿
  
### 投稿機能
* 衣類、本、雑貨の投稿、編集、削除
* 投稿数制限・投稿数拡張
* 投稿画像のプレビュー表示
* 一覧表示・詳細表示
* ページネーション（kaminari）
* タグ付け・タグ検索(acts-as-taggable-on)
* いいね付け(Ajax処理)・いいね数に応じてトレンド一覧表示

### コメント機能
* 投稿に対するコメント、コメント削除(Ajax処理)

### ピックアップ機能
* 他人の投稿を保存(Ajax処理)
* 保存した投稿をマイページに一覧表示

### フォロー機能
* ユーザーのフォロー・フォロー解除
* フォロー中のユーザーとフォロワーの一覧表示
